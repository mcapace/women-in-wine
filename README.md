# Women in Wine — Landing Page

Landing experience for the Wine Spectator **Women in Wine** program. Built with Next.js and Tailwind CSS.

## Setup

```bash
npm install
npm run dev
```

Runs at [http://localhost:4000](http://localhost:4000).

## Build & Deploy

- **Build:** `npm run build`
- **Start:** `npm run start`

Deploy to Vercel via Git: connect the repo in the Vercel dashboard or use `vercel` CLI.

## Structure

- `src/app` — Next.js App Router (layout, page, styles)
- `src/components` — Navigation, Hero, About, ClientShowcase, Footer
- `src/data/clients.ts` — Featured partners (edit names, copy, CTAs here)

## Customization

Update the two featured partners in `src/data/clients.ts`. Replace placeholder names, taglines, descriptions, regions, and CTA links with real client content.
